# Westley's Restocafe Website

A premium, highly-animated restaurant website built with Next.js, Framer Motion, and GSAP.

## Tech Stack
- **Next.js 15 (App Router)**
- **Tailwind CSS**
- **Framer Motion** (UI transitions, tabs, modals)
- **GSAP + ScrollTrigger** (Section reveals, cinematic effects)
- **Lucide React** (Icons)

## Project Structure
- `src/app/`: App router pages and layouts.
- `src/components/`: Modular UI components (Client & Server).
- `src/data/menu.json`: Centralized menu data.

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## Design Decisions
- **Typography**: Playfair Display for headers (upscale), Inter for body (legibility).
- **Colors**: Deep dark palette with gold accents (#C9A84C) and espresso tones.
- **Interactivity**: Mixed Server/Client architecture for optimal SEO and performance.
